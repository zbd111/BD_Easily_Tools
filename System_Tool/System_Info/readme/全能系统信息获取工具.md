# 全能系统信息获取工具

## 1.设计目的

1.此demo希望能一键获取系统信息，减少多次重复的操作GUI读取设备信息，并支持Window/Linux跨平台使用

2.由C/C++语言开发，保证响应速度，并配备使用说明，保证落地可用

## 2.使用说明

此工具支持获取PC设备的 
*时间戳
*系统时间
*主机名称
*系统版本信息
*CPU信息
*GPU(显卡)信息
*内存信息
*电池状态
*磁盘占用
*IP地址
*进程数量

点击相应按钮，即可自动获取当前设备信息到文本框中，并可一键复制

# 4.注意事项

1.设备可能存在多个IP地址，此时获取的是首个IP，因此需要用户自行判断

2.如果使用Linux虚拟机，存在无法获取GPU和电池的情况，之后作者会补充此部分逻辑
